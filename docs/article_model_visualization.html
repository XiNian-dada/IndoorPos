1) 先看全局：算法在做什么

这个模型不是直接回归绝对坐标，而是先学“当前位置相对上一时刻的位移”，所以能更好利用轨迹连续性。

输入
RSSI: \([B,T,N_{AP}]\)
Motion: \([B,T,7]\)
→
时序编码
Top-K token + Frame MLP + TCN
→
双头输出
\(\Delta x,\Delta y\) + grid logits
为什么用增量回归
  • 位移目标更平滑，学习难度低于全局坐标。
  • 可自然加入速度上限约束与轨迹滤波。
  • 对局部 RSSI 抖动更稳。
为什么用 Top-K token
  • RSSI 输入本质是 AP 集合，不是图像。
  • token 保留 AP 身份与强度变化信息。
  • 减少噪声 AP 干扰，降低计算量。
为什么要辅助分类头
  • grid 头提供空间结构监督。
  • 提升主干特征判别性。
  • 最终评价仍以米级误差为主。

6) 公式区（MathJax 渲染 + 纯文本兜底）

正在检测公式渲染状态...

(1) Top-K token 构造
\[ \mathbf{token}_{i,t} = \left[\mathrm{Emb}(AP_{i,t}),\; r_{i,t},\; \left(r_{i,t}-r_{i,t-1}\right),\; \mathrm{rank}_{i,t},\; \mathrm{is\_new}_{i,t}\right] \]
token_{i,t} = [Emb(AP_{i,t}), r_{i,t}, r_{i,t}-r_{i,t-1}, rank_{i,t}, is_new_{i,t}]
(2) 帧编码 + 时序建模
\[ \mathbf{z}_t = f_{frame}(\mathbf{token}_{1:K,t}),\quad \mathbf{h}_{1:T} = \mathrm{TCN}\left([\mathbf{z}_{1:T},\mathbf{m}_{1:T}]\right) \]
z_t = FrameEncoder(tokens_t); h_1:T = TCN([z_1:T, m_1:T])
(3) 双头输出
\[ \Delta\hat{\mathbf{p}}_t = f_{reg}(\mathbf{h}_t),\qquad \hat{\mathbf{g}}_t = f_{cls}(\mathbf{h}_t) \]
delta_pred = Head_reg(h_t); grid_logits = Head_cls(h_t)
(4) 训练损失
\[ \mathcal{L} = \underbrace{\mathrm{SmoothL1}(\Delta\hat{\mathbf{p}}_t,\Delta\mathbf{p}_t)}_{\text{主任务：位移回归}} + \lambda\underbrace{\mathrm{CE}(\hat{\mathbf{g}}_t,\mathbf{g}_t)}_{\text{辅助任务：grid 分类}} \]
L = SmoothL1(delta_pred, delta_gt) + lambda * CE(grid_logits, grid_gt)
(5) 推理递推
\[ \hat{\mathbf{p}}_t = \hat{\mathbf{p}}_{t-1} + \Delta\hat{\mathbf{p}}_t \]
p_t = p_{t-1} + delta_t
(6) 可选后处理
\[ \hat{\mathbf{p}}^{post}_{1:T} = \mathrm{Kalman}\big(\mathrm{SpeedCap}(\hat{\mathbf{p}}_{1:T})\big) \]
p_post = Kalman(SpeedCap(p_raw))

8) 训练/推理流程图（文档友好）

训练阶段
  • 滑窗构造序列样本 \([t-T+1,t]\)。
  • 前向得到 \(\Delta\hat{p}_t\) 与 \(\hat{g}_t\)。
  • 联合损失反传更新参数。
  • 按验证集 mean/rmse 选最优 checkpoint。
推理阶段
  • 实时采样 RSSI + motion。
  • 构造 Top-K token 序列并前向。
  • 输出 \(\Delta\hat{p}_t\) 后递推位置。
  • 可选做速度约束与卡尔曼平滑。
指标解释
  • mean/median/p90/p95/rmse 衡量几何误差。
  • cls_acc 仅反映 grid 辅助头分类能力。
  • 尾误差小而 cls_acc 低并不矛盾。
  • 定位项目优先看米级误差指标。

10) 可直接复制到文档的结论

  1. 该模型核心思想是“集合化 RSSI 编码 + 轻量 TCN 时序建模 + 位移递推定位”。
  2. 相对直接回归绝对坐标，位移建模更符合连续运动规律，误差更稳定。
  3. grid 分类头是辅助监督，不作为最终定位输出，因此 cls_acc 可低于几何精度表现。
  4. 在你当前测试集上，该配置达到 mean=1.274m、rmse=1.493m，属于明显优于其他基线的方案。
